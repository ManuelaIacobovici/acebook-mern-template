# NORMAL START

1. In api
```
JWT_SECRET=SUPER_SECRET npm start
```
2. in frontend
```
npm start
```

# TESTING the Backend (API)

1. in api
```
JWT_SECRET=SUPER_SECRET npm run start:test
```
2. ALSO in api
```
JWT_SECRET=SUPER_SECRET npm run test
```

## TESTING the frontend (React)

1. in api
```
JWT_SECRET=SUPER_SECRET npm run start:test
```
2. in frontend
```
JWT_SECRET=SUPER_SECRET npm start
```
3. Then run the tests in frontend
```
JWT_SECRET=SUPER_SECRET npm run test
```
